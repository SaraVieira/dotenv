dotenv.dev

A collection of developer tools and websites made by Sara Vieira

  • MovieMash - A self hostable movie tracker
  • CSS2JS - Transform CSS to JS Objects, JSX props and anything in between
  • VSCode theme to prism converter - Convert any VSCode theme to a prism theme
  • Calendar Creator - Create links to google calendar events to share them
  • SVG2CSS - Convert SVG images to css background images
  • SVG2JSX - Figma plugins to convert SVG icons to valid JSX
  • GLTFJSX - Convert GLTF files to JSX
  • Table to JSON - Convert a table's HTML to JSON
  • Useful JSON - A set of useful JSON data sets for your projects
  • the_useless_api - An API with useless facts
  • CSS Generators - A list of CSS generators
